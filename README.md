# Real Estate Management System

**Status:** Ongoing Project  
**Tech Stack:** MongoDB, Express.js, React.js, Node.js (MERN)  

---

## Overview

This is a **Real Estate Management System** that allows users to manage and view property listings.  
The system is designed to handle property data efficiently with separate **frontend** and **backend** layers.

---

## Features

- User authentication (Sign Up / Sign In)  
- Add, update, and delete property listings  
- View property details  
- Responsive frontend built with React.js  
- Backend APIs built with Node.js and Express.js  
- Data storage with MongoDB  
- Environment configuration handled with `.env`
